LIKELY QUESTIONS
════════════════════════════════════

Q: What is Reinforcement Learning?
A: RL is a type of ML where an agent
   learns by interacting with environment.
   It takes actions, gets rewards, and
   learns to maximize cumulative reward.
   No labeled data needed!

Q: What is MDP?
A: Markov Decision Process.
   Has 4 components:
   S = State (inventory, days_left)
   A = Action (price level)
   R = Reward (revenue from sale)
   T = Transition (how state changes)

Q: Why DQN over Q-Learning?
A: Q-Learning uses a table to store
   Q-values. Table size grows with
   state space. DQN uses neural network
   instead which scales to any state
   space and can generalize!

Q: What is Experience Replay?
A: Stores experiences in a buffer.
   Samples random mini-batches for
   training. Breaks correlations
   between consecutive experiences
   making training more stable!

Q: What is Target Network?
A: Two networks kept simultaneously:
   Online  → Updated every step
   Target  → Updated every N steps
   Target provides stable training
   signal preventing oscillations!

Q: What did DQN learn?
A: Two key behaviors:
   1. Drops prices near deadline
      to clear remaining inventory
   2. Premium pricing when inventory
      is low (scarcity effect)
   Both proven with charts!

Q: What is Epsilon Greedy?
A: Exploration vs exploitation balance.
   Epsilon = probability of random action.
   Start: ε=1.0 (100% explore)
   End  : ε=0.01 (99% exploit)
   Gradually shifts from random to
   using learned optimal policy!
