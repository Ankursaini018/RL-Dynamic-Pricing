TECHNICAL ACHIEVEMENTS
════════════════════════════════════════

1. MDP FORMULATION:
   State   : (inventory, days_left)
   Actions : 6 price levels ($50-$300)
   Reward  : Revenue per sale
   Penalty : -10 per unsold ticket

2. GYMNASIUM ENVIRONMENT:
   Max Inventory : 50 tickets
   Max Days      : 30 days
   State Space   : 1,581 states
   Gym interface : Fully compliant

3. STOCHASTIC DEMAND:
   Price sensitive (higher=less demand)
   Time sensitive (less days=less demand)
   Stochastic noise for realism

4. BASELINE AGENTS:
   FixedPrice  : Always $150
   Random      : Random each day
   TimeBased   : Adjusts by time left
   DemandBased : Adjusts by inventory
   LinearDecay : Decreases linearly

5. Q-LEARNING AGENT:
   Algorithm : Tabular Q-Learning
   Q-table   : 51×31×6 = 9,486 entries
   Training  : 5,000 episodes
   α (lr)    : 0.10
   γ (disc)  : 0.99
   ε decay   : 1.0 → 0.01
   Result    : Beats best baseline!
